
# Modular Dashboard • Core Shell (Step 1)

This is the minimal core shell + three theme modules (Classic, Dark, Light).

## Run locally
1) Open this folder in VS Code
2) Use **Live Server** (or any static server) to open `index.html`
3) Press **Alt + A** or click **⚙️ Admin** to open Admin
4) Go to **Appearance** tab to switch themes
